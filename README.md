# Mind Spark

Mind Spark is a suite of brain-strengthening web activities designed to help maintain cognitive health. It features a variety of interactive exercises that target different mental skills, from memory and math to focus and relaxation.

## Features

- **Puzzle Games**: Memory Match, Word Scramble, and Color Path.
- **Mental Exercises**: Simple Calc and Focus Challenge (Stroop effect).
- **Wellness**: Guided breathing exercises and a daily journal for recall therapy.
- **Speed & Reaction**: A reflex-testing "bubble pop" game.

## Resources

Mind Spark includes quick access to several important resources:
- Alzheimer's Association
- Lewy Body Dementia Association
- Association for FTD
- Caregiver Action Network
- Crisis Helpline (988)

## How to Run

Simply open [index.html](index.html) in any modern web browser to start using Mind Spark. No installation or server is required.
